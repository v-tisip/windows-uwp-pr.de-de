## <a name="launch-a-uri-from-your-app"></a><span data-ttu-id="996dd-123">Starten eines URIs aus Ihrer App</span><span class="sxs-lookup"><span data-stu-id="996dd-123">Launch a URI from your app</span></span>


<span data-ttu-id="996dd-124">Um die Windows-Karten-App aus Ihrer App zu starten, rufen Sie die [**LaunchUriAsync**](https://msdn.microsoft.com/library/windows/apps/hh701476)-Methode mit einem **bingmaps:**-, **ms-drive-to:**- oder **ms-walk-to:**-URI auf.</span><span class="sxs-lookup"><span data-stu-id="996dd-124">To launch the Windows Maps app from your app, call the [**LaunchUriAsync**](https://msdn.microsoft.com/library/windows/apps/hh701476) method with a **bingmaps:**, **ms-drive-to:**, or **ms-walk-to:** URI.</span></span> <span data-ttu-id="996dd-125">Im folgenden Beispiel wird derselbe URI wie im vorherigen Beispiel gestartet.</span><span class="sxs-lookup"><span data-stu-id="996dd-125">The following example launches the same URI from the previous example.</span></span> <span data-ttu-id="996dd-126">Weitere Informationen zum Starten von Apps mithilfe des URIs finden Sie unter [Starten der Standard-App für einen URI](launch-default-app.md).</span><span class="sxs-lookup"><span data-stu-id="996dd-126">For more info about launching apps via URI, see [Launch the default app for a URI](launch-default-app.md).</span></span>

```cs
// Center on New York City
var uriNewYork = new Uri(@"bingmaps:?cp=40.726966~-74.006076");

// Launch the Windows Maps app
var launcherOptions = new Windows.System.LauncherOptions();
launcherOptions.TargetApplicationPackageFamilyName = "Microsoft.WindowsMaps_8wekyb3d8bbwe";
var success = await Windows.System.Launcher.LaunchUriAsync(uriNewYork, launcherOptions);
```

<span data-ttu-id="996dd-127">In diesem Beispiel wird die [**LauncherOptions**](https://msdn.microsoft.com/library/windows/apps/hh701435)-Klasse verwendet, um sicherzustellen, dass die Windows-Karten-App gestartet wird.</span><span class="sxs-lookup"><span data-stu-id="996dd-127">In this example, the [**LauncherOptions**](https://msdn.microsoft.com/library/windows/apps/hh701435) class is used to help ensure the Windows Maps app is launched.</span></span>
