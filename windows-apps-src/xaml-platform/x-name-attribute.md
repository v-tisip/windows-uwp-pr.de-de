##  <a name="xamlname-grammar"></a><span data-ttu-id="7dc4a-115"> XamlName-Grammatik</span><span class="sxs-lookup"><span data-stu-id="7dc4a-115">XamlName grammar</span></span>

<span data-ttu-id="7dc4a-116">Im Anschluss finden Sie die maßgebende Grammatik für eine Zeichenfolge, die in dieser XAML-Implementierung als Schlüssel verwendet wird:</span><span class="sxs-lookup"><span data-stu-id="7dc4a-116">The following is the normative grammar for a string that is used as a key in this XAML implementation:</span></span>

``` syntax
XamlName ::= NameStartChar (NameChar)*
NameStartChar ::= LetterCharacter | '_'

DecimalDigit ::= '0'-'9'
CombiningCharacter::= none
```
