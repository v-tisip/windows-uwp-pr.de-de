### <a name="an-example-staticresource-usage"></a><span data-ttu-id="5d07b-145">{StaticResource}-Beispielverwendung</span><span class="sxs-lookup"><span data-stu-id="5d07b-145">An example {StaticResource} usage</span></span>

<span data-ttu-id="5d07b-146">Der folgende XAML-Beispielcode stammt aus dem [XAML-Datenbindungsbeispiel](http://go.microsoft.com/fwlink/p/?linkid=226854).</span><span class="sxs-lookup"><span data-stu-id="5d07b-146">This example XAML is taken from the [XAML data binding sample](http://go.microsoft.com/fwlink/p/?linkid=226854).</span></span>

```xml
<StackPanel Margin="5">
    <!-- Add converter as a resource to reference it from a Binding. --> 
    <StackPanel.Resources>
        <local:S2Formatter x:Key="GradeConverter"/>
    </StackPanel.Resources>
    <TextBlock Style="{StaticResource BasicTextStyle}" Text="Percent grade:" Margin="5" />
    <Slider x:Name="sliderValueConverter" Minimum="1" Maximum="100" Value="70" Margin="5"/>
    <TextBlock Style="{StaticResource BasicTextStyle}" Text="Letter grade:" Margin="5"/>
    <TextBox x:Name="tbValueConverterDataBound"
      Text="{Binding ElementName=sliderValueConverter, Path=Value, Mode=OneWay,  
        Converter={StaticResource GradeConverter}}" Margin="5" Width="150"/> 
</StackPanel> 
```

<span data-ttu-id="5d07b-147">In diesem bestimmten Beispiel wird ein Objekt erstellt, hinter dem eine benutzerdefinierte Klasse steht. Dieses Objekt wird als Ressource in einer [**ResourceDictionary**](https://msdn.microsoft.com/library/windows/apps/br208794) erstellt.</span><span class="sxs-lookup"><span data-stu-id="5d07b-147">This particular example creates an object that's backed by a custom class, and creates it as a resource in a [**ResourceDictionary**](https://msdn.microsoft.com/library/windows/apps/br208794).</span></span> <span data-ttu-id="5d07b-148">Dieses `local:S2Formatter`-Element muss auch einen **x:Key**-Attributwert haben, um als gültige Ressource zu gelten.</span><span class="sxs-lookup"><span data-stu-id="5d07b-148">To be a valid resource, this `local:S2Formatter` element must also have an **x:Key** attribute value.</span></span> <span data-ttu-id="5d07b-149">Der Wert des Attributs wird auf "GradeConverter" festgelegt.</span><span class="sxs-lookup"><span data-stu-id="5d07b-149">The value of the attribute is set to "GradeConverter".</span></span>

<span data-ttu-id="5d07b-150">Die Ressource wird dann etwas weiter hinten im XAML angefordert, wo `{StaticResource GradeConverter}` angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="5d07b-150">The resource is then requested just a bit further into the XAML, where you see `{StaticResource GradeConverter}`.</span></span>

<span data-ttu-id="5d07b-151">Die Syntax der {StaticResource}-Markuperweiterung legt eine Eigenschaft einer anderen Markuperweiterung ([{Binding}-Markuperweiterung](binding-markup-extension.md)) fest, sodass zwei geschachtelte Markuperweiterungssyntaxen vorhanden sind.</span><span class="sxs-lookup"><span data-stu-id="5d07b-151">Note how the {StaticResource} markup extension usage is setting a property of another markup extension [{Binding} markup extension](binding-markup-extension.md), so there's two nested markup extension usages here.</span></span> <span data-ttu-id="5d07b-152">Die innere wird zuerst ausgewertet, damit die Ressource zuerst abgerufen wird und als Wert verwendet werden kann.</span><span class="sxs-lookup"><span data-stu-id="5d07b-152">The inner one is evaluated first, so that the resource is obtained first and can be used as a value.</span></span> <span data-ttu-id="5d07b-153">Dieses Beispiel wird auch in der {Binding}-Markuperweiterung gezeigt.</span><span class="sxs-lookup"><span data-stu-id="5d07b-153">This same example is also shown in {Binding} markup extension.</span></span>
