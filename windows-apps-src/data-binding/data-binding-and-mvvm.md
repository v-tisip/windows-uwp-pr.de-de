# <a name="data-binding-and-mvvm"></a><span data-ttu-id="79d09-104">Datenbindungen und MVVM</span><span class="sxs-lookup"><span data-stu-id="79d09-104">Data binding and MVVM</span></span>

<span data-ttu-id="79d09-105">Model-View-ViewModel (MVVM) ist ein UI-Architektur-Muster für die Benutzeroberfläche und nicht-UI-Code entkoppeln.</span><span class="sxs-lookup"><span data-stu-id="79d09-105">Model-View-ViewModel (MVVM) is a UI architectural design pattern for decoupling UI and non-UI code.</span></span> <span data-ttu-id="79d09-106">Das MVVM deklarativ definieren die Benutzeroberfläche in XAML und verwenden bindungsmarkup Daten für die Verbindung zu anderen Ebenen, Daten und Befehle enthält.</span><span class="sxs-lookup"><span data-stu-id="79d09-106">With MVVM, you define your UI declaratively in XAML and use data binding markup to link it to other layers containing data and commands.</span></span> <span data-ttu-id="79d09-107">Data-Binding-Infrastruktur bietet eine Kopplung, die die Benutzeroberfläche hält, und die verknüpften Daten synchronisiert und leitet Benutzereingaben für die entsprechenden Befehle.</span><span class="sxs-lookup"><span data-stu-id="79d09-107">The data binding infrastructure provides a loose coupling that keeps the UI and the linked data synchronized and routes user input to the appropriate commands.</span></span>
