<span data-ttu-id="5d26f-107">Beispielsweise würden Sie den folgende JSON-Code verwenden, damit Wasserflächen in Rot, Wasserbeschriftungen in Grün und Landflächen in Blau dargestellt werden:</span><span class="sxs-lookup"><span data-stu-id="5d26f-107">For example, you'd use the following JSON to make water areas appear in red, water labels appear in green, and land areas appear in blue:</span></span>

```json
        "elements":{"vegetation":{"fillColor":{"value":"#999999","ignoreTransform":true}}}
    }
```
