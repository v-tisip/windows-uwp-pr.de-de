        "elements":{"water":{"fillColor":"#FF0000","labelColor":"#00FF00"}}

So wird der exakte bereitgestellte Wert mithilfe der IgnoreTransform-Eigenschaft verwenden.</span><span class="sxs-lookup"><span data-stu-id="9d6d0-112">This behavior can be turned off, thereby using the precise provided value, by using the ignoreTransform property.</span></span>

        "elements":{"vegetation":{"fillColor":{"value":"#999999","ignoreTransform":true}}}
    }
```
