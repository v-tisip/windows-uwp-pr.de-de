<span data-ttu-id="6eb06-105">Microsoft-Mapping-Technologien verwenden _Karten-Stylesheets_ , um die Darstellung von Karten definieren.</span><span class="sxs-lookup"><span data-stu-id="6eb06-105">Microsoft mapping technologies use _map style sheets_ to define the appearance of maps.</span></span>  <span data-ttu-id="6eb06-106">Eine Karte dem Stylesheet wird mithilfe von JavaScript Object Notation (JSON) definiert und kann in verschiedenen u. a. in einer Windows Store-Anwendung [MapControl](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapcontrol) über die [MapStyleSheet.ParseFromJson](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapstylesheet.parsefromjson#Windows_UI_Xaml_Controls_Maps_MapStyleSheet_ParseFromJson_System_String_) -Methode verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="6eb06-106">A map style sheet is defined using JavaScript Object Notation (JSON) and can be used in various ways including in a Windows Store application's [MapControl](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapcontrol) through the [MapStyleSheet.ParseFromJson](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapstylesheet.parsefromjson#Windows_UI_Xaml_Controls_Maps_MapStyleSheet_ParseFromJson_System_String_) method.</span></span>

<span data-ttu-id="6eb06-107">Stylesheets können interaktiv mit der [Karte Formatvorlagen-Editor](https://www.microsoft.com/p/map-style-sheet-editor/9nbhtcjt72ft) -Anwendung erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="6eb06-107">Style sheets can be created interactively using the [Map Style Sheet Editor](https://www.microsoft.com/p/map-style-sheet-editor/9nbhtcjt72ft) application.</span></span>

        "settings":{"landColor":"#0000FF"},
        "elements":{"water":{"fillColor":"#FF0000","labelColor":"#00FF00"}}

<span data-ttu-id="6eb06-109">Diese JSON kann verwendet werden, um alle Beschriftungen und Punkte auf einer Karte entfernen.</span><span class="sxs-lookup"><span data-stu-id="6eb06-109">This JSON can be used to remove all labels and points from a map.</span></span>

```json

    {"version":"1.*", "elements":{"mapElement":{"labelVisible":false},"point":{"visible":false}}}
```

<span data-ttu-id="6eb06-110">In einigen Fällen wird der Wert einer Eigenschaft transformiert, um das endgültige Ergebnis zu erzielen.</span><span class="sxs-lookup"><span data-stu-id="6eb06-110">Sometimes the value of a property is transformed to produce the final result.</span></span>  <span data-ttu-id="6eb06-111">Beispielsweise hat die Pflanzen-FillColor je nach Typ der Entität geringfügig andere Töne.</span><span class="sxs-lookup"><span data-stu-id="6eb06-111">For example, vegetation fillColor has slightly different shades depending on type of the entity being displayed.</span></span>  <span data-ttu-id="6eb06-112">Dieses Verhalten kann deaktiviert werden. So wird der exakte bereitgestellte Wert mithilfe der IgnoreTransform-Eigenschaft verwenden.</span><span class="sxs-lookup"><span data-stu-id="6eb06-112">This behavior can be turned off, thereby using the precise provided value, by using the ignoreTransform property.</span></span>

```json
    {"version":"1.*",
        "settings":{"shadedReliefVisible":false},
        "elements":{"vegetation":{"fillColor":{"value":"#999999","ignoreTransform":true}}}
    }
```
